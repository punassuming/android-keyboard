# Purpose

FUTO Keyboard exists to provide a modern Android keyboard that prioritizes privacy, offline operation, and user control.

## Core goals

- Stay usable without requiring cloud connectivity
- Avoid spyware-like behavior and unnecessary data exposure
- Support practical multilingual and layout customization workflows
- Maintain a transparent, auditable codebase forked from AOSP LatinIME

## Scope in this repository

This repository contains the Android app implementation, resources, translation integration points, native components, tests, and build automation for stable/unstable/playstore flavors.
