# Collaboration

## Where collaboration happens

- Issues and pull requests: [GitHub mirror](https://github.com/futo-org/android-keyboard/)
- Primary source hosting: [internal GitLab](https://gitlab.futo.org/keyboard/latinime)
- App information and downloads: [FUTO Keyboard website](https://keyboard.futo.org/)

## Team workflows reflected in the repo

- CI definitions are maintained in `.gitlab-ci.yml`
- Translation synchronization is scripted in `syncTranslations.sh`
- Release/signing setup in CI uses `setUpPropertiesCI.sh`

## Translation collaboration

Community translation contributions are handled via Pontoon:

- https://i18n-keyboard.futo.org/
