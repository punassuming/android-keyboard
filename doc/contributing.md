# Contributing

## Before opening a pull request

1. File or find an issue on GitHub when applicable.
2. Prepare your changes in a dedicated branch.
3. Run relevant local builds and checks for your change.

## Pull requests and license requirements

- Pull requests are accepted via GitHub: [https://github.com/futo-org/android-keyboard/](https://github.com/futo-org/android-keyboard/)
- Because of the custom license, contributors must sign the [CLA](https://cla.futo.org/) after opening a PR.

## Layout-specific contributions

Keyboard layouts are maintained separately in:

- [https://github.com/futo-org/futo-keyboard-layouts](https://github.com/futo-org/futo-keyboard-layouts)

The layouts repository contributions do not require the CLA because they are Apache-2.0.
